## RedAQ Forge
A Linux electron-based(pain) launcher for RedAQ.

### Compiling and running this works for Node.js v12 and above (I think).

## Compiling
```sh
# NPM
npm i
npm run make64 # or make32, if you have a 32-bit system

# YARN:
yarn
yarn make64 # or make32
```

## Running
```sh

npm run start

-- or --

yarn start
```
